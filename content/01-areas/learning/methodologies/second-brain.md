# Building a Second Brain

These are the notes for [[01-areas/personal/index|personal development]] and the Building a Second Brain methodology.

## Key Concepts

### Building a Second Brain (BASB)

A methodology for creating a personal knowledge management system using digital tools.

#### Core Principles

- **Capture** - Keep what resonates
- **Organize** - Save for actionability
- **Distill** - Find the essence
- **Express** - Show your work

#### Connection to Personal System

This knowledge management approach supports both learning and project work by creating clear pathways from information consumption to knowledge creation.
